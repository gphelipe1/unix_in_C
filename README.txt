----------- I N S T R U Ç Õ E S --------------

º   Para compilar o programa em um terminal linux basta executá-lo
    normalmente através do comando "gcc main.c -o exe" e em seguida
    executar o programa no arquivo executável gerado, utilizando "./exe"



º   Para fazer a listagem LS basta digitar o comando "ls". Para utilizá-lo com
    a estrutura GREP basta digitar no formato: ls | grep _string_de_busca_



º   Para chamar o comando PWD, basta digitar "pwd. Para utilizá-lo com
    a estrutura GREP, basta digitar no formato: pwd | grep _string_de_busca_



º   Para usar o comando CD, basta digitar "cd" e em seguida digitar a 
    pasta desejada.
        Ex: cd /home/usuario/Documentos
        Ex  cd .. (volta para o diretório anterior)
        Ex  cd ~ (vai para a pasta home)
        Ex  (já na pasta home) cd usuario 



º   Para utilizar o comando MORE e ler arquivos, basta digitar "more" 
    e em seguida digitar o arquivo desejado no diretório atual.
        EX: more arquivo.txt
    Para limitar a saída do arquivo com o GREP, basta digitar 
    no formato: more arquivo.txt | grep _string_de_busca_



 ~~~~ ~~~~ ~~~~ Extras ~~~~ ~~~~ ~~~~
º   Para sair do programa, basta digitar "exit" ou "EXIT"

º   Para limpar a tela basta digitar "clear"
